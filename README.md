# qdr.nvim

## Requirements

`fzf` binary in `$PATH`
lyaml lua module

## Installation

```
use 'vijaymarupudi/nvim-fzf'
use 'ZacxDev/qdr.nvim'
```

## Usage

```vimscript
" Open the qdr picker
:Qdr
```

## Example qdr.yml
```yml
Run Go Generate: go generate ./...
Run Tests: go test ./...
```
